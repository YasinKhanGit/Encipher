Three Image format

JPEG
JPEG is a commonly used image format that supports millions of colors. It is best for photographs but does not support transparency.
PNG
PNG supports high-quality images with transparency. It is ideal for logos, icons, and graphics that need clear edges.
SVG
SVG is a vector-based format. It can be scaled to any size without losing quality and is used for icons, logos, and illustrations.
